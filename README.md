# Lotus AI 汉化补丁

基于 SPT Lotus 模组的 AI 辅助中文翻译。

## 安装

1. 确保已安装 **Lotus 本体** 和 **Virtual's Custom Quest Loader**
2. 备份 `user/profiles/` 下的存档
3. 将本补丁的 `user` 文件夹复制到 SPT 根目录覆盖
4. 启动服务器验证，莲花的第一个任务在 **5 级** 解锁

## 版本对应

| 汉化版本 | Lotus 版本 | SPT 版本 |
|---------|-----------|---------|
| v1.0.0 | 1.7.1 | 4.0.13 |
